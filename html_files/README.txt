HTMLファイルの配置方法

==========================================
クイックスタート
==========================================

1. ブラウザでページを開く
   https://yutura.net/tag/4662/

2. 右クリック → 「ページのソースを表示」

3. すべて選択してコピー（Ctrl+A → Ctrl+C）

4. page1.html として保存

5. ページ2以降も同様に繰り返す
   - page2: https://yutura.net/tag/4662/?p=2
   - page3: https://yutura.net/tag/4662/?p=3
   ...

6. すべて保存したら実行:
   cd 1_scraping
   python batch_html_parser.py

==========================================
ファイル名のルール
==========================================

必ず page1.html, page2.html, page3.html... の順番で命名してください

データがないページは空のままでOK（自動でスキップされます）

==========================================
詳細手順
==========================================

【ページ1の場合】
1. ブラウザでページを開く
   https://yutura.net/tag/4662/

2. ページ上で右クリック → 「ページのソースを表示」
   （または Ctrl+U）

3. すべて選択（Ctrl+A）

4. コピー（Ctrl+C）

5. html_files/page1.html をメモ帳で開く

6. すべて選択（Ctrl+A）

7. 貼り付け（Ctrl+V）

8. 保存（Ctrl+S）

9. 同じ手順を page2.html, page3.html... でも繰り返す

==========================================
トラブルシューティング
==========================================

Q: HTMLをコピーしたのにデータが取得できない
A: ページのソース（HTML）ではなく、表示画面をコピーしている可能性があります。
   必ず「ページのソースを表示」からコピーしてください。

Q: 何ページまで取得すればいい？
A: データがなくなるまで！空のページは自動でスキップされます。

Q: ファイル名を間違えた
A: page1.html, page2.html, page3.html... の順番で命名してください。
